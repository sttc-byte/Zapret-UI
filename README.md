# 🛡️ Zapret-UI v5.0 

**Zapret-UI** — это высокопроизводительная графическая оболочка (GUI) для управления движком `zapret`. Основная задача проекта — обеспечить комфортное использование инструментов обхода блокировок без необходимости взаимодействия с консольными окнами и ручного редактирования конфигурационных файлов.

* **Разработчик UI:** [.sttc (sttc-byte)](https://github.com/sttc-byte)
* **Оригинальный движок:** [bol-van/zapret](https://github.com/bol-van/zapret)
* **Рекомендуемая сборка:** [Flowseal](https://github.com/Flowseal/zapret-discord-youtube)

---

> [!IMPORTANT]
> **Важный момент:** Это только «лицо» (интерфейс). Внутри нет самого движка обхода. Тебе нужно скачать готовую сборку (рекомендую от Flowseal) и просто положить мой экзешник рядом с папкой `zapret`.

### 🚀 Ключевые особенности v5.0
* **Stealth Mode:** Запуск всех процессов `winws.exe` в скрытом режиме. Полное отсутствие лишних окон в панели задач.
* **Intelligent Tuner:** Встроенный сканер конфигураций. Автоматически перебирает доступные пресеты, проверяет доступность ресурсов (Discord/YouTube) и выбирает лучший вариант на основе минимального пинга.
* **UAC-Bypass Autostart:** Реализация автозагрузки через **Windows Task Scheduler**. Это позволяет программе запускаться с наивысшими правами администратора без вызова окна подтверждения UAC при каждом входе в систему.
* **Modern UI Engine:** Интерфейс построен на базе **Eel (Chromium/Edge)**. Поддержка аппаратного ускорения, эффектов размытия (Mica/Acrylic) и полной кастомизации визуальной составляющей.
* **Data Integrity:** Атомарная система сохранения настроек исключает повреждение файла `config.json` при критических сбоях или экстренном завершении работы ОС.

---

### ⚙️ Совместимость и ОС
* **Windows 10 / 11:** Кастомизация тем, цветов и установка фона работают на обеих системах.
* **Только для Windows 11:** Функции «Невидимая рамка» и «Цвет рамки» созданы специально под API Windows 11.
    * _Примечание:_ Функции находятся в бета-тесте и могут работать нестабильно из-за особенностей рендеринга DWM.

### 📦 Установка и запуск
1. Скачайте актуальный релиз **Zapret-UI.exe**.
2. Подготовьте папку с бинарными файлами обхода (рекомендуется сборка [Flowseal](https://github.com/Flowseal/zapret-discord-youtube)).
3. Убедитесь, что папка с батниками называется `zapret`.
4. Разместите `Zapret_UI_v5.0.exe` в корневой директории (рядом с папкой `zapret`).
5. Запустите приложение **от имени Администратора**.

### 🛠️ FAQ / Решение проблем

**— Почему в Диспетчере задач виден Microsoft Edge?**
Программа построена на базе движка **Eel (Chromium/WebView2)**. Наличие процессов `msedgewebview2` необходимо для работы графического интерфейса. Это нормальное поведение.

**— Антивирус ругается на файл?**
Это стандартная реакция на софт, упакованный через PyInstaller и работающий с сетью. Файл чист, просто добавьте его в исключения.

**— Статус «Активен», но сайты не открываются?**
* Отключите протокол **QUIC** в браузере (`chrome://flags/#enable-quic` -> **Disabled**).
* Используйте встроенный **Сканер**, чтобы подобрать рабочий конфиг именно под вашего провайдера.
* Если ничего не помогает, изучите [официальные фиксы от Flowseal](https://github.com/Flowseal/zapret-discord-youtube#частые-проблемы-и-их-решения).
